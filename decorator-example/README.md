# TypeScript decorators

<https://blog.logrocket.com/a-practical-guide-to-typescript-decorators/#:~:text=Automatic%20error%20guard>

Do this once:

```bash
npm install -g typescript
```

and then:

```bash
tsc example.ts
node example.js
```

(Note: enter `.ts` not `.js`)

or just do this:

```bash
npm run go
```
